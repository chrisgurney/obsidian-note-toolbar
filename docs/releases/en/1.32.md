# What's new in Note Toolbar v1.32?

_Make sure you've updated to the latest version to get all of these changes._

## Fixes 🛠️ 

### Obsidian Community Review Feedback [#546](https://github.com/chrisgurney/obsidian-note-toolbar/issues/546)

This release addresses the majority of the issues raised on the new [Obsidian community scorecard](https://community.obsidian.md/plugins/note-toolbar#scorecard).

Please comment on [#546](https://github.com/chrisgurney/obsidian-note-toolbar/issues/546) if you have any questions or concerns.

### Other fixes

- API: `ntb.toolbar` was being immediately removed after being triggered by a script.
