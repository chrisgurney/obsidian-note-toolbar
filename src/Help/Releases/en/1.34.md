# What's new in Note Toolbar v1.34?

_Make sure you've updated to the latest version to get all of these changes._

## Improvements 🚀

### Toolbar icon + description

You can now optionally add an icon and description to toolbars, to help you find and manage them better. Use the gear icon (⚙️) next to the toolbar name field to set them.
  - Search by description in settings and in [Quick Toolbars](https://github.com/chrisgurney/obsidian-note-toolbar/wiki/Quick-Tools#quick-toolbars-window).
  - Icons appear: on floating buttons, in the tab bar, on the file menu (if enabled), in Quick Toolbars, and in the toolbar list in settings.
  - The icon is suggested as a default for item menus (if no icon is already set).
  - The [Note Toolbar CLI](https://github.com/chrisgurney/obsidian-note-toolbar/wiki/Note-Toolbar-CLI) shows descriptions (using the command `note-toolbar verbose`).
  - _Thank you @moyf for [the suggestion](https://github.com/chrisgurney/obsidian-note-toolbar/issues/329)!_

### Other improvements

- [Quick Toolbars](https://github.com/chrisgurney/obsidian-note-toolbar/wiki/Quick-Tools#quick-toolbars-window): Now shows toolbar previews.

## Fixes 🛠️

- Bases/Canvas: `Editor width` style was not using the full editor width.
- Settings: Various settings UI fixes for Obsidian 1.13.
